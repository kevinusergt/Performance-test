# Student management

## Description
This is a python program which allows managing the information of enrolled students.

## Simple logic

* The menu is displayed.
* The user's selection is received.
* The corresponding function is called.

## Main Features

1. Add student.
2. Show list.
3. Search student.
4. Delete student.
5. Update student.
6. Exit

## Functions
* `Menu()` -> Shows available options.
* `register_students()` -> Add a new student to the list.
* `fetch()` -> Muestra los productos del inventario.
* `consult()` -> Search for and return to the searched student.
* `update_student()` -> Replace an existing student.
* `delete_student()`-> Delete a student from the list.
